# Simpson's Rule
n次に拡張したシンプソンの公式です。

と思っていましたが、ニュートン・コーツの公式と呼ぶそうです。既にあったみたいですね。

defineの値とfunctionを自由に編集できます。functionはmpfrを使用してください。結構大変です。


分割幅を$h$とした$x_N \cdots x_{N+n}$の$n+1$点による$f(x)$の$n$次式近似は、
$$
\int_{x_N}^{x_{N+n}} f(x )\,dx = nh\sum_{i = 0}^{n} \alpha_i f(x_{n+i})
$$
$$
\alpha_i = \sum_{k = 1}^{n+1} \sum_{l = k}^{n+1} \frac{(-1)^{i+l+1}i^{l-k}n^{k-1}{n+1\brack l}}{k(n-i)!i!}
$$
として表されます。ただし${n\brack m}$は第一種スターリング数です。
手計算なので、どこか間違っていたら教えてください。

これをc言語でmpfrを使用しコードを作成しました。分割数とmpfrのbit数が編集でき、計算精度を任意に調整できます。$sin(10x)$について精度を調べた画像を参考までに添付しておきます。
